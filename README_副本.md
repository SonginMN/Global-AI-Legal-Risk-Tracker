# Global AI Legal Risk Tracker

## 项目简介

本项目是一个全球AI法律风险追踪系统，定期收集、整理和分析全球范围内与人工智能相关的：
- **立法与法规更新 (Legislation & Regulations Updates)**
- **诉讼与执法动态 (Litigation & Enforcement)**

## 工作节奏

| 项目 | 时间 |
|------|------|
| 定期更新 | 每周一，北京时间 10:00 |
| 首次检索 | 2026年1月16日（周五） |

### 更新流程

```
1. 每周一 10:00 → 用户触发："更新本周报告"
2. AI 检索动态 → 生成报告 → 更新本地文件
3. 用户本地预览 → http://localhost:8080
4. 用户确认无误 → AI 提供文件包
5. 用户推送到 GitHub → 网站更新
```

> 💡 **为什么选择周一？** AI法规动态通常在工作日发布，周一更新可以覆盖上周全部动态，且正好赶上团队工作周开始。

## 优先级体系

### 🔴 最高优先级 (P0) - 深度分析
**欧盟和美国有关进出口管制、监管制裁的信息**
- 出口管制法规（Export Control）
- 经济制裁（Sanctions）
- 技术转让限制（Technology Transfer）
- 实体清单（Entity List）更新
- 跨境数据传输限制
- AI芯片/半导体出口管制

### 🟠 高优先级 (P1)
- 重大AI立法生效或修订
- 高额罚款/处罚案例
- 里程碑式判决
- 涉及主要AI公司的诉讼

### 🟡 标准优先级 (P2)
- 常规法规更新
- 草案/征求意见稿
- 一般性执法动态

### 深度分析内容（针对P0）
1. 法规原文解读
2. 影响范围评估
3. 合规时间表
4. 违规后果分析
5. 同业应对措施参考
6. 具体行动建议

## 信息来源

主要追踪的地区和监管机构：

### 立法/法规
- **美国**: 联邦法规、各州法律（加州、德州、科罗拉多州等）、BIS出口管制、OFAC制裁
- **欧盟**: EU AI Act、GDPR相关、出口管制、制裁
- **中国**: 网络安全法、AI相关法规、国家标准
- **英国**: Online Safety Act、Data (Use and Access) Act
- **亚太**: 韩国、日本、新加坡、越南等

### 诉讼/执法
- 美国FTC调查
- 欧盟竞争法执法
- 各国数据保护机构行动
- 版权侵权诉讼
- 隐私与安全相关案件

详见 `sources.md` 文件

## 交付内容

每周交付包括：
1. **信息要点表格**: 结构化的法规/诉讼信息
2. **来源链接**: 所有信息附有效可访问的原始链接
3. **分析总结报告**: 趋势分析、风险评估、应对建议
4. **P0深度分析**: 针对进出口管制/制裁的详细解读
5. **归档文件**: JSON格式的结构化数据，便于后续查询

## 设计规范

### 核心原则
1. **亮色主题**：使用米白色背景，避免深色风格
2. **交互一致性**：不可点击的元素不做可点击样式
3. **官方链接**：所有法规必须提供官方法条链接，而非新闻报道
4. **内容完整**：分析报告需包含重点提醒、趋势分析、风险建议、官方链接汇总

详细设计规范见 `issues/log.md` 中的「设计规范与验证要求」章节。
